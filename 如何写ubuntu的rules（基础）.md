- 1.系统按照字母表顺序读取rules文件，如果想要被优先读取，可以以0开头命名
- 2.rules文件中有两种关键词：匹配关键词使用==，分配关键词使用=。
- 3.匹配关键词指用户不可更改的东西，如KERNEL(设备在内核中的名字)和ATTR(设备的属性)等；分配关键词指用户可以更改的东西，如NAME(设备在/dev下的名字)和MODE(设备权限)等
- 4.SYMLINK使用+=，是设备的链接或叫别名，我们一般使用SYMLINK，不修改NAME。也可以设置多个链接`SYMLINK+="cdrom cdrom0"`
- 5.可以使用命令`udevadm info -a -p $(udevadm info -q path -n /dev/video0)`查询设备信息，这里是摄像头。本机输出如下

- 7.例子
```
# :=表示之后不可更改
SUBSYSTEMS=="usb", ATTRS{idProduct}=="4920", ATTRS{idVendor}=="2304", MODE:="0666", SYMLINK+="my_camera"
# 或者
KERNEL=="video0", KERNELS=="1-10:1.0", MODE:="0666", SYMLINK+="my_camera"
```
